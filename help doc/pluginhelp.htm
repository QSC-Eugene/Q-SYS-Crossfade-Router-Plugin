Crossfade Router

A drop in replacement for a Router Component that utilizes internal crossfaders.

Index

  • Properties
  • Controls
  • Control Pins

Properties

Input Count

Number of inputs on the router.

Type: integer

Default Value: 8

Min: 2

Max: 128

Output Count

Number of outputs on the router.

Type: integer

Default Value: 8

Min: 1

Max: 128

Type

Type of router. Can be Mono, Stereo or Multi-channel.

Type: enum

Default Value: Mono

Choices:

  • Mono
  • Stereo
  • Multi-channel

Count

Number of channels per input or output when Type is set to Multi-channel.

Type: integer

Default Value: 8

Min: 2

Max: 64

Show Debug

Show debug messages in the console

Type: Boolean

Default Value: false

Controls

crossfade.time

Float Knob

The time it takes to crossfade between inputs in seconds.

Min: 0.01

Max: 100

crossfade.type

Text Control

The type of crossfade to use.

fading.n

LED Indicator

Indicates that the output is fading.

mute.n

Toggle Button

Mutes the output.

output.[output].input.[input].select

Toggle Button

Selects a specific input to be crossfaded to the corresponding output.

select.n

Integer Knob

Selects the input to be crossfaded to the corresponding output.

Min: 1

Max: Number of outputs
